# Simple CRM

A simple CRM demo application built with Angular and Supabase. This project was created for learning purposes and demonstrates common Angular concepts such as components, services, routing, and data binding.

## Teck Stack

- Angular 21
- Angular Material
- Supabase (Database & Realtime)
- TypeScript

## Features

- View all customers in a table
- Add new customers
- Edit customer details
- Delete customers
- Realtime updates

## Getting Started

### Prerequisites

- Node.js
- Angular CLI

### Installation

1. Clone the repository
   `git clone https://github.com/your-username/simple-crm.git`

2. Install dependencies
   `npm install`

3. Add your Supabase credentials in environment.ts
   export const environment = {
   supabaseURL: 'YOUR_SUPABASE_URL',
   supabaseKey: 'YOUR_SUPABASE_KEY',
   };

4. Start the development server
   `ng serve`
